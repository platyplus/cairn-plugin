# cairn — Claude plugin

_Generated from `plugins/claude/` in the Cairn monorepo — do not edit here; edit the source._

Drive Cairn from Claude — design data collections, import and capture records, and explore them as tables, charts, and maps. Schema-driven, offline-first data for humanitarian and field teams.

## Install

```
/plugin marketplace add platyplus/cairn-plugin
```

Then enable the **cairn** plugin. It bundles the Cairn skills and the
hosted Cairn connection (https://cairn.platyplus.io).
